## 📝 UI设计规范

### 主题配色
| 用途 | 色值 |
|------|------|
| 背景 | `#0B0C10` (极微弱深蓝紫倾向的高级暗色) |
| 主色调 | `#dc2626` (红色-600) |
| 强调色 | `#f97316` (橙色) |
| 文字 | `#ffffff` (白色) / `#ffffffb3` (70%透明度) |
| 边框 | `rgba(255,255,255,0.1)` (10%透明度) |

### 按钮样式
- 高饱和渐变：`bg-gradient-to-r from-red-600 to-rose-600`
- 内阴影：`shadow-red-900/50`
- 悬浮光泽：`hover:brightness-110`
- 活动缩放：`active:scale-95`

### 组件样式
- 卡片圆角：`rounded-xl` (12px)
- 按钮渐变：`from-red-600 to-red-700`
- 背景模糊：`backdrop-blur-xl`
- 悬停效果：`hover:scale-105`
